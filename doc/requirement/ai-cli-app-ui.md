## 需求
这个是Agent 的CLI 前端页面展示。基本布局样式和Claude Code CLI APP一样。

### 技术栈
UI 框架：textual
彩色输出：Textualize/rich
语音:Python

### UI 要求
#### 大体布局
彩色的字符画欢迎语（当前CLI APP 版本，软件名称，使用贴士，现在使用的模型）
聊天记录区域（做好空处理,要有彩色输出，特别是代码块）
输入框（支持斜线指令，用户输入了'/'就要有下拉框做输入提醒和tab完成补全）

### 配置
如果检测到用户在用户目录的.mini-code-agent 文件夹里面的config配置文件不存在，用户输入命令启动后就要引导用户填写AI供应商名字，下拉选择接口格式（目前只支持openAI和anthropic的接口格式），请求地址，API 密钥。用户输入完成写到用户目录的.mini-code-agent 文件夹里面的config配置文件。
如果存在配置文件直接跳过。
注意：配置文件路径有两种情况：
1. 用户目录下的.mini-code-agent 文件夹里面的config配置文件
2. 当前路径项目的.mini-code-agent 文件夹里面的config配置文件
